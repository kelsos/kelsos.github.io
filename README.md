# Personal Website

Can be found under [kelsos.net](https://kelsos.net).

Built with:

- [Nuxt 4](https://nuxt.com)
- [@nuxt/content](https://content.nuxt.com)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Inter](https://rsms.me/inter/) (variable font)

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
